So I've had a few ideas for board games I could make in Godot. The problem is that I don't really know that much about making games in Godot, especially board games. But after following YouTuber BornCG's very helpful 2D and 3D platformer tutorials to learn the basics of Godot, I realized that he has another one on how to make Memory. I decided that it would be a good idea to follow that tutorial and see what I learn.

My review

Overall, the tutorial was quite effective and taught me a lot about Godot. It could get a bit overwhelming or boring at times, but that's nothing a healthy break can't fix. While it's designed for Godot 3.2, I was using Godot 4.x and found it mostly still worked. Overall, I highly recommend it to anyone, whether kid or adult, looking to learn Godot 3.2. If you're trying to learn Godot 4.x, this tutorial will still get the job done if you're patient.

Here's a list of topics this tutorial covers.

  • Singletons
  • Text and Labels
  • Text Styling
  • Arrays
  • Buttons
  • Basic Object management
  • Textures
  • Control Nodes
